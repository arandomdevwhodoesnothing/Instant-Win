# Instant Win

Instant Win is a simple static web page that displays a celebratory **"You Won!"** message with clean styling and a subtle animation.

## Overview
This project is built using basic HTML and CSS, keeping things lightweight and easy to understand. It’s ideal as a fun demo, placeholder page, or starting point for small web projects.

## Usage
You can view the page by visiting:  
https://arandomdevwhodoesnothing.github.io/Instant-Win/

(This is not a live demo, just the hosted page location.)

## Files
- `index.html` – The main HTML structure
- `style.css` – Styling and animation
